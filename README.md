# 📌 Project Overview

AtmoSync AI is a complete Data Analytics and Business Intelligence project designed to monitor temperature-sensitive products transported through a cold chain.

The system simulates IoT sensor data, performs automated data cleaning, feature engineering, spoilage risk analysis, financial loss estimation, recommendation generation, and exports the processed data to Power BI dashboards.

The project demonstrates an end-to-end analytics pipeline similar to real logistics systems used by pharmaceutical, dairy, seafood, food, and vaccine companies.

---

# 🎯 Objectives

- Simulate IoT cold-chain sensor data
- Generate realistic shipment datasets
- Clean and preprocess raw data
- Perform feature engineering
- Calculate spoilage risk
- Estimate financial loss
- Generate business recommendations
- Produce executive business insights
- Export analytics-ready data
- Build an interactive Power BI Dashboard

---

# 🏢 Business Problem

Cold-chain logistics companies transport products that require controlled environmental conditions.

Examples include:

- Vaccines
- Medicines
- Dairy Products
- Seafood
- Frozen Foods
- Fruits
- Vegetables

Small environmental changes can cause product spoilage, resulting in significant financial losses.

This project helps detect those risks early.
